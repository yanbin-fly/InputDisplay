## 1. 简介

**InputDisplay** 是一款运行在 Windows 11 上的输入监控工具。它以轻量的浮窗形式覆盖在桌面和其他应用之上，实时记录并展示你所有的键盘按键和鼠标操作。

### 主要用途

- **教学演示**：在屏幕录制或直播中，向观众展示你的每一个操作步骤
- **习惯分析**：了解自己在日常使用中最频繁使用的快捷键和按键
- **调试辅助**：在测试软件或游戏时，记录操作的时序和频率
- **操作备忘**：回顾某个时段内的完整操作历史

### 系统要求

| 项目 | 要求 |
|------|------|
| 操作系统 | Windows 11 或 Windows 10 |
| 运行时 | .NET 8.0 Runtime（[点击下载](https://dotnet.microsoft.com/download/dotnet/8.0)） |
| 磁盘空间 | 约 10 MB |
| 内存 | 约 30 MB |

## 2. 安装

### 2.1 方式一：直接运行（推荐）

1. 下载 `InputDisplay.exe` 文件
2. 双击运行即可，无需安装

> 如果提示需要 .NET 运行时，请先安装 [.NET 8.0 Runtime](https://dotnet.microsoft.com/download/dotnet/8.0)。

### 2.2 方式二：自包含版本

如果你不想安装 .NET 运行时，可以下载标有 `self-contained` 的版本，该版本已包含运行时环境，下载后直接双击运行即可。

### 2.3 验证安装

运行程序后，屏幕上应该出现一个浅色半透明的浮窗，位于屏幕右下角区域。

## 3. 快速入门

### 3.1 启动程序

双击 `InputDisplay.exe`，浮窗将出现在屏幕右下角，标题栏显示"● 录制中"，表示已开始监控。

### 3.2 开始监控

程序启动后自动开始监控，不需要任何额外操作：

- 按下键盘任意键 → 浮窗记录并显示该按键
- 点击鼠标 → 浮窗记录点击类型和位置坐标
- 滚动滚轮 → 浮窗记录滚动方向和幅度

### 3.3 查看统计

点击浮窗右上角的 **📊** 按钮，展开统计面板，可以查看：
- 键盘按键总次数
- 鼠标点击总次数
- 各按钮点击次数
- 高频按键 Top5
- 当前会话时长

再次点击 **📊** 按钮可收起统计面板。

### 3.4 隐藏浮窗

- 点击 **─**（最小化）按钮：浮窗隐藏，仅在托盘区显示图标
- 点击 **✕**（关闭）按钮：浮窗隐藏，继续在后台运行监控

### 3.5 恢复浮窗

双击屏幕右下角系统托盘中的程序图标，即可恢复浮窗显示。

## 4. 界面介绍

### 4.1 整体布局

```
┌──────────────────────────────────────────────────┐
│  输入监控   ● 录制中           📊  │ ─ │ ✕ │   ← 标题栏（可拖拽）
├──────────────────────────────────────────────────┤
│  ⌨  当前输入                                │  ← 当前输入卡片
│     Ctrl+V                       14:32:15   │    （有新输入时显示）
├──────────────────────────────────────────────────┤
│  📈 输入统计                                │  ← 统计面板
│  键盘按键:  142      鼠标点击:  23       │    （点击📊展开）
│  高频键 Top5:  C×18  V×15  X×12 ...     │
├──────────────────────────────────────────────────┤
│  ⌨ Ctrl+C                                    │
│  ⌨ Ctrl+V                                    │  ← 历史记录列表
│  🖱 左键按下 @ (420, 315)                   │
│  ⌨ CapsLock                                  │
│  ⌨ Esc                                       │
└──────────────────────────────────────────────────┘
```

### 4.2 各区域说明

#### 标题栏

| 元素 | 说明 |
|------|------|
| 输入监控 | 标题文字 |
| ● 录制中 | 绿色标签，表示正在监控 |
| 📊 | 展开/收起统计面板 |
| ─ | 最小化到托盘 |
| ✕ | 隐藏窗口（后台继续运行） |

> **提示**：标题栏可以拖拽，移动浮窗位置。

#### 当前输入卡片

当有新的输入时，卡片显示当前正在输入的内容：

- **蓝色卡片**（⌨ 图标）：表示键盘输入
- **绿色卡片**（🖱 图标）：表示鼠标操作
- 无输入时，卡片自动隐藏

#### 历史记录列表

滚动显示最近的输入历史，每条记录包含：

| 内容 | 说明 |
|------|------|
| ⌨ / 🖱 | 输入类型图标 |
| Ctrl+C | 格式化后的输入描述 |
| 14:32:15 | 时间戳 |

## 5. 功能详解

### 5.1 键盘输入监控

程序捕获并显示所有键盘按键，包括：

| 按键类型 | 显示示例 |
|----------|----------|
| 普通字母键 | `A`, `B`, `C` |
| 数字键 | `1`, `2`, `3` |
| 功能键 | `F1`, `F2`, `...`, `F12` |
| 组合键 | `Ctrl+C`, `Ctrl+Shift+A`, `Alt+Tab` |
| 修饰键单独按 | `LCtrl`, `RAlt`, `LWin` |
| 特殊键 | `Enter`, `Esc`, `Space`, `Backspace`, `Tab` |
| 方向键 | `↑`, `↓`, `←`, `→` |
| 小键盘 | `Num1`, `Num2`, `Num+`, `Num-` |

**关于大小写**：普通字母键显示为大写（如 `A`），与是否按住 Shift 无关。这是 Windows 虚拟键码的标准行为。

### 5.2 鼠标操作监控

程序记录以下鼠标事件：

| 操作 | 显示示例 | 说明 |
|------|----------|------|
| 左键按下 | `左键按下 @ (420, 315)` | 坐标为屏幕绝对坐标 |
| 左键释放 | `左键释放 @ (420, 315)` | |
| 右键按下 | `右键按下 @ (420, 315)` | |
| 中键按下 | `中键按下 @ (420, 315)` | 滚轮按键 |
| 滚轮滚动 | `滚轮 1格 ↑` | ↑ 上滚，↓ 下滚 |
| 横滚轮滚动 | `横滚 1格 →` | → 右滚，← 左滚 |
| 鼠标移动 | `移动 @ (420, 315)` | |

> **注意**：鼠标移动事件在统计面板中合并计数，不在历史列表中逐条显示（避免刷屏）。

### 5.3 统计面板

点击 **📊** 按钮展开统计面板：

| 统计项 | 说明 |
|--------|------|
| **键盘按键** | 会话内键盘按键总次数 |
| **鼠标点击** | 会话内鼠标点击总次数 |
| **鼠标移动** | 会话内鼠标移动总次数 |
| **滚轮滚动** | 会话内滚轮滚动总次数 |
| **左/中/右键** | 各按钮的点击次数 |
| **高频按键 Top5** | 使用频率最高的 5 个按键及次数 |
| **会话时长** | 当前监控会话的持续时间 |

**统计重置**：右键托盘图标 → 选择「重置统计」，可清空所有统计数据并重新开始计时。

### 5.4 复制历史条目

双击历史列表中的任意一条记录，可以将该条记录复制到剪贴板。便于记录特定操作的描述。

### 5.5 窗口位置记忆

关闭程序后，下次启动时浮窗会自动出现在上次关闭时的位置（屏幕分辨率变化时会有边界保护）。

## 6. 系统托盘

### 6.1 托盘图标

程序运行时，系统托盘（屏幕右下角）会出现一个应用图标。图标说明：

| 状态 | 托盘图标 |
|------|----------|
| 运行中（浮窗可见） | 标准应用图标 |
| 运行中（浮窗已隐藏） | 标准应用图标 |

### 6.2 右键菜单

右键点击托盘图标，弹出菜单：

| 菜单项 | 功能 |
|--------|------|
| **显示窗口** | 恢复浮窗显示 |
| **重置统计** | 清空所有统计数据，重新计时 |
| **退出** | 完全关闭程序（退出后台监控） |

### 6.3 气泡通知

点击「最小化」或「关闭」按钮时，会弹出气泡通知提示：
- 最小化："已最小化到托盘，双击托盘图标恢复"
- 关闭："窗口已隐藏，运行于后台。继续监控中..."

## 7. 常见问题

### Q1：程序需要管理员权限吗？

**A**：通常不需要。但如果你使用的是某些安全软件或沙盒环境，可能会拦截全局钩子。建议将程序添加到安全软件的白名单中。

### Q2：程序会被杀毒软件报毒吗？

**A**：因为程序使用了 Windows 全局钩子（`SetWindowsHookEx`），这是键盘记录类软件的常规技术，部分杀毒软件可能会误报。建议将程序添加到白名单。本程序不收集、不保存、不上传任何数据。

### Q3：为什么关闭浮窗后程序还在运行？

**A**：这是设计行为。点击「✕」按钮仅隐藏浮窗，监控功能继续在后台运行。如需完全退出，请右键托盘图标 →「退出」。

### Q4：为什么 CapsLock 单独显示，而不是影响字母大小写？

**A**：程序只记录按键事件，不干涉系统的键盘状态。CapsLock 键本身作为独立按键被记录，其切换大小写的功能不受影响。

### Q5：统计面板的数据会保存吗？

**A**：当前版本统计面板数据仅保存在内存中，程序退出后自动清零。每次启动是独立的新会话。

### Q6：如何设置开机自启动？

**A**：
1. 右键 `InputDisplay.exe` →「创建快捷方式」
2. 按 `Win+R`，输入 `shell:startup`，回车
3. 将快捷方式粘贴到打开的文件夹中

### Q7：程序会影响游戏或全屏应用吗？

**A**：不会。浮窗设置为"最顶层但不抢占焦点"（`Topmost="True"`），不会干扰其他应用的操作。但透明浮窗会轻微占用屏幕空间。

## 8. 卸载

### 8.1 停止程序

1. 右键系统托盘图标
2. 选择「退出」

### 8.2 删除文件

直接删除 `InputDisplay.exe` 及相关文件即可完成卸载。

程序不向系统注册表写入任何数据（除了可选的开机自启动快捷方式），不产生额外的配置文件（如已创建位置记忆文件，可同时删除）：

```
%LOCALAPPDATA%\InputDisplay\settings.txt
```

### 8.3 卸载 .NET 运行时（可选）

如果你不需要其他 .NET 程序，可以从「设置 → 应用 → 已安装的应用」中卸载「.NET 8.0 Runtime」。
